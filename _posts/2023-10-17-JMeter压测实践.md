# Non-GUI

通过命令方式启动并生成报告：`jmeter -n -e -f -t xxx.jmx -l xxx.jtl -o report`

- -n：以非图形化界面的形式运行
- -e：在脚本运行结束后生成 HTML 报告
- -f：强制覆盖已存在的结果文件和报告
- -t：测试文件路径（.jmx）
- -l：结果文件路径（.jtl）
- -o：测试报告输出目录路径

```bash
运行压测命令：
jmeter -n -t callcenter.jmx -l result.jtl -e -o /Users/jiaqi.zhang/Desktop/xt/20231016ST/result
```
