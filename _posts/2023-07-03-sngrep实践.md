# 安装

环境:CentOS

安装步骤：

```text
yum copr enable irontec/sngrep
yum install sngrep
```

# 使用

实时捕获 SIP 信令：

```shell
sngrep -d eth0 -O save.pcap port 5060 and udp
```

查看信令文件：

```shell
sngrep -I file.pcap
```
